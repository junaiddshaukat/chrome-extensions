### Manual Installation (Developer Mode)

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top right)
4. Click **Load unpacked**
5. Select the `issues-reminder-extension` folder

## Usage

1. Click the extension icon in your toolbar
2. Enter a repository in `owner/repo` format (e.g., `facebook/react`)
3. Click **Add**
4. You'll receive notifications when new issues are created!

### Tips

- Track popular repos with active issue discussions
- Look for repos with `good first issue` labels
- Be quick to respond when you get a notification!

## Limitations

- **Rate Limits**: Uses GitHub's unauthenticated API (60 requests/hour per IP). If tracking many repos, you may hit this limit.
- **Public Repos Only**: Cannot track private repositories without authentication.
- **Polling-Based**: Checks every minute, so there may be a slight delay.
